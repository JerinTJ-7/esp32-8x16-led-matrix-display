# ESP32 8×16 LED Matrix Display Board

## Overview

This project is a custom PCB-based LED matrix display designed using KiCad.

The system uses:
- ESP32 DevKit V1
- 74HC595 Shift Registers
- LED Matrix (8×16)
- Blynk IoT
- Wokwi Simulation

## Features

- Text display
- Scrolling messages
- Wi-Fi connectivity
- Mobile control using Blynk
- Custom KiCad PCB

## Tools Used

- KiCad 9
- Arduino IDE
- Wokwi Simulator
- Blynk IoT
